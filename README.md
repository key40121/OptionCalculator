## Building the project and running the test
```bash
cmake -S . -B build
cmake --build build

cd build/test
./main.out
```

## Prerequisites
C++ Compiler - needs to support at least the C++20 standard, i.e. MSVC, GCC, Clang  
CMake v3.22+
